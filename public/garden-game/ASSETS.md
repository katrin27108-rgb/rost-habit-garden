# 3D assets used by the local garden game slice

- `Willow_4.obj` / `Willow_4.mtl`: Quaternius, Ultimate Nature Pack, CC0.
- `shrub_03.gltf` and included files: Poly Haven, Shrub 03 by Rico Cilliers, CC0.
- `HVGirl.glb`: Babylon.js official sample asset, used only in this local technical prototype; replace with the application's commissioned character before publishing.
- `babylonjs.loaders.min.js`: Babylon.js loader package, Apache 2.0.

The generated open-garden image is not used as the playable world. The rendered environment is real-time 3D geometry.
